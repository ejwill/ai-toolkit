# MakerWorld Listing

## When to use

Use this skill when writing MakerWorld titles, descriptions, README-style documentation, tags, compatibility tables, or other public-facing model copy.

## Workflow

1. Lead with the model's purpose and fit.
2. Explain the key features and customization path.
3. Add compatibility, print notes, and troubleshooting only as needed.
4. Keep the copy readable and focused on the buyer's decision.

## Core Rules

- Write the listing for buyers first, maintainers second.
- Start with what the model is and what problem it solves.
- Explain compatibility, size presets, and print mode choices clearly.
- Keep README or customization docs technical and useful.
- Use tags that match the product family, print method, and part numbers.
- Keep public copy concise and avoid repeating the same advice in multiple places.

## Common Checks

- The title and opening paragraph are clear at a glance.
- The compatibility table matches the actual model presets.
- The README or guide explains the settings users will actually see.
- Print notes match the model's real orientation and assembly needs.
- Tags are specific, not filler.
